# 如何找到想做的事

作者：八木仁平，译者：徐艺菊

## 第一章 阻碍找到想做的事的五种误区

| 误区                             | 事实                                           |
| -------------------------------- | ---------------------------------------------- |
| 必须是一生坚持的事情             | 做“现在想做的事”就可以了                       |
| 找到想做的事时会有命中注定的感觉 | 即使找到想做的事情，一开始也是处于感兴趣的阶段 |
| 必须是对别人有益的事             | 为自己而活也是在帮助别人                       |
| 必须多行动才能找到               | 了解自己才能找到想做的事                       |
| 想做的事不能成为工作             | 想做的是在自己心中，实现手段在社会中           |
